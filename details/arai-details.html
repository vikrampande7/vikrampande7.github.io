Solar Panel Cleaning Robot

Designed and built an autonomous solar panel cleaning robot using C++, Arduino, Ultrasonic Sensors, and Motor control. The robot navigated solar panels to remove dust and debris, improving panel efficiency without manual intervention.

  • Programmed Arduino microcontroller in C++ to control motor movement and sensor-based navigation
  • Integrated ultrasonic sensors for obstacle detection and edge avoidance
  • Implemented motor control logic for precise traversal across panel surfaces
